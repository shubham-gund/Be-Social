# Social Media Application (Frontend)

## Overview
This project is a feature-rich social media application frontend, enabling users to connect, share, and interact in real time. Built with React, React Query, and Socket.io, the app offers an intuitive user experience and incorporates modern technologies for seamless functionality. Integrated with Gemini AI, users can chat with an AI assistant for enhanced engagement. The app also includes light and dark modes for a personalized user experience.

---

## Features

### Core Functionality
- **Post Photos and Thoughts**: Share your photos and thoughts with others.
- **CRUD Operations**:
  - Create, Read, Update, and Delete posts effortlessly.
  - Update your profile picture and cover photo.
  - Modify user information like bio, name, and more.
  - Change passwords securely.
- **Real-Time Notifications**: Get instant updates for likes, comments, and other interactions using React Query.
- **Real-Time Messaging**: Communicate with friends using Socket.io for real-time chat functionality.
- **AI-Powered Chat**: Engage with Gemini AI for meaningful conversations.

### Additional Features
- **Light and Dark Modes**: Toggle between light and dark themes for better accessibility and user preference.
- **Responsive Design**: Optimized for all screen sizes.
- **Interactive UI**: Smooth transitions and animations for better user experience.
- **User Authentication**: Ensure data security and privacy.

---

## Tech Stack

### Frontend
- **React**: For building the UI components.
- **React Query**: For efficient state management and real-time notifications.
- **Socket.io**: For enabling real-time messaging.
- **Gemini AI Integration**: For AI-powered user interaction.
- **Daisy UI & Tailwind CSS**: For styling and a customizable, modern design.

### Tools and Libraries
- **Axios**: For API calls.
- **React Router**: For seamless navigation.

## Installation

### Prerequisites
- Node.js installed on your system.
- A backend server to handle API requests and Socket.io connections.

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/social-media-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd social-media-app
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
5. Open your browser and navigate to `http://localhost:5173`.
